Autogenerated navigation

Since Source v. 0.3.2 complex navigation output is available using data-nav attribute.
Relative and absolute paths supported.

Embedding

Common scheme of embedding looks like this: <div class="source_catalog" data-nav="docs"> <ul class="source_catalog_list"> <!--Navigation generated dynamically via Global Navigation module--> <img src="/core/i/process.gif" alt="Loading..."/> </ul> </div>

Required attribute data-nav names the content to be displayed.

Headings and description auto output

Basically, info.json is used to display title and additional information, if any. This data can be alternatively provided in the markup:

<div class="source_catalog" data-nav="docs"> <ul class="source_catalog_list"> <h2 class="source_catalog_title">Nice clarify plugin</h2> <div class="source_catalog_tx">Custom description about that</div> <!--Navigation generated dynamically via Global Navigation module--> <img src="/core/i/process.gif" alt="Loading..."/> </ul> </div>
